# VSS Autoencoder Training

This repository contains a Jupyter Notebook adapted for Kaggle that can be used to train and evaluate different autoencoder architectures on the image reconstruction problem for a VSS (Very Small Size Soccer) simulator dataset.

The notebook supports experiments with:

* Convolutional Autoencoders (CAEs)
* Variational Autoencoders (VAEs)
* Sparse Autoencoders (SAEs)

The implementation includes configurable training settings, reconstruction evaluation and visualization utilities for comparative experiments.
